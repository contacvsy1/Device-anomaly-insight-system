Execution

How to run the pipeline locally

Run the project from the repository root. The script creates required folders if they do not already exist.

Setup Commands

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/du_anomaly_insight.py

Expected Console Output

When the script runs successfully, it prints:

Data summary

Raw data sample, anomaly label counts, disk utilization summary, and anomaly type counts.

Insight preview

Injected spike detection count, top anomaly insight rows, and file paths for saved outputs.

Generated Files

File Created By Meaning
data/du_data.csv raw_data.to_csv(...) Complete simulated telemetry dataset.
outputs/du_anomaly_insights.csv insights.to_csv(...) Only rows detected as anomalies, enriched with reasoning and action recommendations.

Troubleshooting

Symptom Likely Cause Fix
Module import error Dependencies were not installed into the active environment. Activate .venv and rerun pip install -r requirements.txt.
No output files The script did not complete or was run from an unexpected setup. Run python src/du_anomaly_insight.py from the project root and check console errors.
Different row values Seed, row count, dependencies, or code changed. Confirm seed=42 and unchanged simulation defaults.
